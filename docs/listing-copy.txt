APP NAME:
FocusFlow — Hard-Enforcement App Blocker for Android

DEVELOPER / AUTHOR:
TBTechs

SHORT DESCRIPTION (≤ 150 characters):
The hardest-to-bypass Android app blocker. Accessibility Service + VPN + Device Admin. Temptation Log. Free.

LONG DESCRIPTION:
FocusFlow is a free, open-source Android app blocker built on the premise that willpower shouldn't be required to use an app blocker. Once a focus session starts, blocked apps genuinely cannot be opened — there is no dismiss button, no "ignore limit" tap-through, and no workaround that doesn't require factory-resetting your phone.

FocusFlow combines three independent enforcement layers:

1. Android Accessibility Service — detects blocked apps within 300 ms and overlays a non-dismissible block screen. Re-checks every 300 ms for 5 cycles to close mid-activity escape windows.

2. Local Null-Routing VPN — routes blocked apps' network traffic to an on-device tunnel that never forwards packets externally. In PER_APP mode, only blocked apps lose internet. In GLOBAL mode, all traffic is cut except emergency calls. Zero bytes leave your device.

3. Device Admin + Session PIN — Device Admin prevents uninstalling FocusFlow via Settings. System Guard intercepts navigation to Accessibility Settings, Clear Data, and App Info during sessions. A SHA-256 hashed session PIN is stored at the native Android layer and cannot be bypassed even if the JavaScript bridge is disabled.

UNIQUE FEATURES (not found in any other free app blocker):
- Temptation Log: records every blocked-app attempt with app name and timestamp (up to 500 entries)
- Weekly Temptation Report: push notification every Sunday with 7-day blocked attempt totals by app
- Content-specific blocking: block YouTube Shorts without blocking YouTube; block Instagram Reels without blocking Instagram
- Aversive feedback: screen dims + vibration + sound triggers the moment a blocked app opens
- Sideload Guard: apps installed during an active session are automatically blocked
- Clock tamper detection: validates session timestamps on startup to prevent the "change system time" bypass
- Boot Recovery: BootReceiver restores active sessions after device reboot

OTHER FEATURES:
- Scheduled Greyout Windows (calendar-based blocking, no active session required)
- Daily Allowance Engine (count/time budget/interval per app)
- Launcher Replacement (minimal home screen with only allowed apps)
- Pomodoro Mode (auto-cycling work and break sessions)
- Keyword Blocker (blocks by on-screen text, URL, and search terms)
- Backup & Restore (.focusflow file format)
- 7 languages: English, Chinese (Simplified), French, German, Japanese, Korean, Spanish
