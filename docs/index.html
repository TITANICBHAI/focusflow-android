Enforcement Layers

Five independent blockers. All active at once.

Each layer works independently. Disable one and the other four still hold. This is not a timer app with a lock screen on top.

01

AccessibilityService

Intercepts every window-change event instantly using Android's AccessibilityService API. The moment a blocked app gains focus, it is pushed away — before the user can interact with it.

02

Null-Routing VPN

An on-device VPN tunnel routes all traffic for blocked apps to a null interface — completely cutting network access without any external server, without root, and without modifying system hosts.

03

Device Administrator Mode

Android Device Admin prevents FocusFlow from being uninstalled through Settings during an active session. Removing it requires the session to end first — which requires the SHA-256 PIN.

04

System Guard

Monitors and intercepts navigation to the Accessibility Settings screen and the app's Clear Data screen during active sessions. Common workarounds that disable other blockers are themselves blocked.

05

SHA-256 Session PIN

All stop-session operations are gated by a native SHA-256 hashed PIN — computed in the native layer. Even if the JavaScript bridge were bypassed, the hash check still runs in native code.

Features

Built beyond basic blocking

Content-level blocking, aversive feedback, weekly reports, 30+ OEM brands supported. Completely free.

Content-Level Blocking

Blocks specific content within apps — YouTube Shorts, Instagram Reels — not just the entire app. The rest of the app remains accessible.

Session Recovery on Boot

Active sessions survive device reboots. All five enforcement layers are re-activated automatically at boot — no way to escape via restart.

Weekly Temptation Report

A weekly digest shows which blocked apps were attempted most, peak distraction hours, and session completion rates. Awareness, quantified.

Aversive Feedback

Optional haptic and visual disruption when a blocked app is attempted. Not a pleasant notification — a deliberate friction point.

30+ OEM Brands

Tested and tuned for quirks across Samsung, Xiaomi, OnePlus, Oppo, Vivo, Realme, and 25+ more. OEM-specific AccessibilityService behaviors handled per-brand.

Completely Free

No subscription. No premium tier. No ads. Every enforcement layer is available to every user from the first install.
